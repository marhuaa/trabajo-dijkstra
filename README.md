# Optimización de Rutas Nacionales (Perú) - Algoritmo de Dijkstra

Este repositorio contiene un proyecto académico integral desarrollado para la carrera de **Ingeniería de Sistemas**. El proyecto combina fundamentos teóricos de Inteligencia Artificial (IA) y Ética con una implementación práctica de Investigación de Operaciones utilizando el Algoritmo de Dijkstra para la navegación terrestre en el Perú.

---

##  Parte 2: Implementación Práctica (Dijkstra)

La aplicación permite calcular la ruta más corta entre las 25 capitales de departamento de Perú, visualizando el trayecto dinámicamente sobre un mapa.


###  Instalación y Uso

1. **Instala las librerías necesarias:**
   ```bash
   pip install streamlit folium streamlit-folium
´´´
2. **Ejecuta la aplicación:**

```Bash
streamlit run dijkstra.py
```
Interacción:

Selecciona la ciudad de Origen (ej. Puno).

Selecciona la ciudad de Destino (ej. Lima).

Haz clic en "Encontrar Camino" para trazar la ruta en el mapa y ver la distancia total en KM.
